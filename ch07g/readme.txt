Developer: Sherwin John Calleja Tragura
Status: Prototype stage
Date finished: February 15, 2022

=========================== Description ====================================

This prototype is an auction system that accepts any products from auctioneers 
for bidding. The application should determine the highest price bid to sell a 
product. The prototype is secured using various approaches to securing 
FastAPI microservice applications.

This application uses the FastAPI framework driven by Python 3.8.
This application now involves data storage using PostgreSQL 13. 
The backup database schema is in the /database folder of the project.

The requirements.txt will guide you with the dependencies. Just install by running:
pip install -r requirements.txt
